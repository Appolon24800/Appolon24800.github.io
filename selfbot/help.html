Commands & Features:

  • help:

    Send you to this website

  • clear: <amount>

    Clear messages (ex: clear 10)

  • leave:

    Leave the current group

  • webhook:

    Just create a webhook if you have the perm

  • delwebhook: <webhook>

    Delete a webhook

  • time:

    Give you the current time

  • date:

    Give you the current date

  • block:

    Block a user (the user with which you are in dm)

  • user: <discordid>

    Give you information about a user only with their ID (user UserID)

  • nitro

    Generate and send a random generated nitro lonk

  • info:

    Give you information about the selfbot

  • fact:

    Give you a random fact

  • ipinfo: <ip>

    Give you information about an IP address

  • gender:

    Give you the gender of someone

  • shutdown:

    Exit the selfbot

  • Nitro sniper

    Snipe nitro that are send in some servers

  • Giveaway joiner

    Automatically join giveaway from certain bot
