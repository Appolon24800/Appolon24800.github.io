Commands:

  • Help:

    Make this message

  • Ping:

    Give you your ping (in ms)

  • Clear:

    Clear messages (ex: clear 10)

  • Site:

    Send a normal link

  • Site2:

    Send a link that looks like code (with this `)

  • Site3:

    Send the link of a website without its embed

  • Legit:

    Your grabber link will be sent

  • Web:

    Your website link will be sent

  • Leave:

    Leave the current group

  • Webhook:

    Just create a webhook if you have the perm

  • Time:

    Give you the current time

  • Date:

    Give you the current date

  • Block:

    Block a user (the user with which you are in dm)

  • User:

    Give you information about a user only with their ID (user UserID)

  • sslist:

    Create a .txt file and save all your servers (the name and the vanity link if the server has one)

  • togglerpc:

    Toggle / Toggle-off the discord RPC

  • dnd:

    Toggle / Toggle-off the Do Not Disturb mode

  • fact:

    Give you a random fact

  • ipinfo:

    Give you information about an IP address

  • gender:

    Give you the gender of someone

  • backup:

    Create files with your account information

  • lunarRPC:

    Remove the lunar client RPC (github.com/Fuwn/soyuz)
